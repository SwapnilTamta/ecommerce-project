# E-commerce Website

This is a simple e-commerce website built using HTML, CSS, JavaScript, and JSON.

## Features

*   User can view products
*   User can search products
*   User can add products to cart
*   User can view cart
*   User can checkout


## Technologies Used

*   HTML
*   CSS
*   JavaScript
*   JSON
